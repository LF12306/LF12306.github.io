多选题

31. 膜分离技术的特点有( )
32. 亲和层析的结合方式包括( )

34. 膜污染的种类包括( )
35. 膜材料的选择应考虑的因素有( )
36. 膜分离过程中,影响通量的因素有( )

判断题

37. 膜污染是指在膜分离过程中,由于料液中组分在膜表面或膜孔内的吸附、沉积、浓缩等作用,导致膜通量下降和分离性能恶化的现象。( )
38. 离子交换层析是根据分子大小进行分离的。( )
39. 盐析是利用蛋白质在不同盐浓度下的溶解度差异进行分离的方法。( )
40. 超滤主要用于分离溶液中的离子和低分子量物质。( )
41. 亲和层析是一种高效、高选择性的分离技术。( )
42. 酶工程中,酶的分离纯化是关键步骤之一。( )
43. 膜通量是指单位时间内通过单位膜面积的溶液体积。( )
44. 浓差极化会导致膜通量下降。( )
45. 膜分离技术只适用于液体的分离。( )
46. 反渗透膜能够截留几乎所有的溶解性盐类和有机物。( )
47. 微滤膜的孔径通常大于超滤膜。( )
48. 膜污染可以通过增加操作压力来缓解。( )
49. 离心分离是利用离心力使混合物中不同密度的组分分离。( )
50. 凝胶过滤层析适用于分离分子量相近的蛋白质。( )
51. 膜分离过程中的跨膜压差是推动力。( )
52. 纳滤膜主要用于去除大分子物质和微生物。( )
53. 膜清洗是膜分离过程中的重要环节。( )
54. 渗透压是膜分离的推动力。( )
55. 超滤膜的截留分子量范围一般在1000至1000000道尔顿之间。( )
56. 膜分离是一种物理分离过程。( )
57. 蛋白质的等电点是指其净电荷为零时的pH值。( )
58. 膜组件的膜装填密度越高越好。( )
59. 离子交换剂的选择取决于目标分子的电荷性质。( )
60. 亲和层析是基于生物分子间特异性结合的原理进行分离的。( )
61. 膜污染是膜分离过程中不可避免的现象。( )
62. 反渗透膜对盐的截留率很高。( )
63. 膜分离过程的能耗通常高于传统分离方法。( )
64. 凝胶过滤层析中,大分子物质先被洗脱出来。( )
65. 膜分离技术可以用于生产无菌水。( )
66. 膜分离过程通常在常温下进行。( )
67. 蛋白质的盐析是由于盐离子竞争水分子,导致蛋白质溶解度下降。( )
68. 吸附作用的最根本因素是吸附质和吸附剂之间的作用力,也就是范德华力,它是一组分子引力的总称,具体包括三种力:定向力、诱导力和色散力。( )
69. 吸附等温线表示最高吸附量,并可用来推断吸附剂结构、吸附热和其它理化特性。( )
70. 样品极性较小、难以分离者,吸附剂用量可适当提高至样品量的10~20倍。( )
71. 凝胶过滤层析中,分子直径比凝胶孔径小的分子,即进入凝胶相内,延长了路径,向下移动的速度超过于大分子。( )
72. 结晶过程具有成本低、设备简单、操作方便,广泛应用于氨基酸、有机酸、抗生素、维生素等产品的精制。( )
73. 固定化酶或细胞可利用酶工程或细胞工程技术将酶或细胞固定于不溶性载体上或限制于一定区域内,使其能够重复使用。( )
74. 工业上通常采用低分子量的葡聚糖凝胶制备蛋白质。( )
75. 破碎细胞的常用方法是物理法、化学法、酶法。( )
76. 微生物细胞常用的分离方法有离心分离和膜分离。( )
77. 沉淀分离是利用组分溶解度差异或沉降速度差异进行分离的方法。( )
78. 离心机的分离效率只与离心力有关。( )

主观题

79. 吸附作用的最根本因素是吸附质和吸附剂之间的作用力,也就是范德华力,它是一组分子引力的总称,具体包括三种力:定向力、诱导力和色散力。
